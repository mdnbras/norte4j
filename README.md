# 🌎 Norte4J

Site oficial da comunidade **Norte4J**, um grupo voltado para desenvolvedores apaixonados por tecnologia, especialmente do ecossistema **Java, Kotlin e backend moderno**.

## 🚀 Sobre o projeto

Este repositório contém o código-fonte do site da comunidade Norte4J.

O objetivo do site é:
- Divulgar eventos e meetups
- Compartilhar conteúdo técnico
- Fortalecer a comunidade de desenvolvedores
- Centralizar informações da Norte4J

## 🛠️ Tecnologias utilizadas

- Vite / React

## 📦 Como rodar o projeto

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/norte4j.git

# Acesse a pasta
cd norte4j

# Instale as dependências
npm install

# Execute o projeto
npm run dev
